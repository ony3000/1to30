# 1to30

> 한 때 즐겨 했던 [1부터 50까지](./original-app.png)를 기리며 만드는 **1부터 30까지**.<br>
> 2017년 8월에 저장소 없이 날것으로 작성했던 버전을 다시 만들기.

## Requirements

- Node 16
- [pnpm v7](https://pnpm.io/)

## Getting Started

```bash
# install dependencies
pnpm install

# serve with hot reload at localhost:17030
pnpm run dev
```
